# UX Writing 스타일 변환기

텍스트를 입력하면 5가지 UX Writing 스타일로 자동 변환해주는 웹 애플리케이션입니다.

## 스타일

- **😊 친절한 버전**: 따뜻하고 격려하는 톤
- **😎 캐주얼한 버전**: 친구 같은 편한 톤 (해요체)
- **🤝 정중한 버전**: 표준 존댓말 (일반적인 서비스 톤)
- **🎩 경어 버전**: 공식적이고 정중한 경어
- **📝 단어형**: 명사/키워드 위주로 간결하게

## 설치 및 실행

### 1. 의존성 설치
```bash
npm install
```

### 2. API 키 설정
`.env.example`을 참고하여 `.env` 파일을 생성하고 Gemini API 키를 설정합니다:

```bash
cp .env.example .env
```

그 후 `.env` 파일을 열어 `GEMINI_API_KEY`를 설정합니다.

**API 키 생성:**
1. https://makersuite.google.com/app/apikey 접속
2. "Create API key" 클릭
3. API 키 복사하여 `.env` 파일의 `GEMINI_API_KEY` 값으로 설정

### 3. 서버 시작
```bash
npm start
```

서버가 시작되면 http://localhost:3000 에서 웹앱을 사용할 수 있습니다.

## 사용 방법

1. 왼쪽 패널의 텍스트 입력 칸에 변환할 텍스트를 입력합니다
2. 오른쪽 패널에 5가지 스타일로 변환된 결과가 실시간으로 표시됩니다
3. 각 결과 우측의 "복사" 버튼으로 클립보드에 복사할 수 있습니다

## 기반 가이드라인

이 도구는 다음 UX Writing 가이드라인을 참고하여 만들어졌습니다:

- **토스**: 캐주얼한 해요체, 능동형, 긍정형 커뮤니케이션
- **삼성**: 단순성, 솔직성, 신뢰도 있는 명확함
- **구름**: 친절하고 지지하는 톤, 친근하면서도 전문적

## 기술 스택

- **프론트엔드**: HTML, CSS, Vanilla JavaScript
- **백엔드**: Node.js, Express
- **AI**: Google Gemini API

## 개발 메모

- Gemini API를 사용하여 정교한 변환을 수행합니다
- 프론트엔드는 별도 서버 없이 로컬에서도 기본 변환 기능을 제공합니다
- API 키가 없어도 기본 규칙 기반 변환이 작동합니다
